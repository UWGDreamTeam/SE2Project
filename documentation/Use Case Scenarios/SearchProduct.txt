Context - User is within the product tab page of the inventory page

Primary Flow - 
1 - User inputs a string in the search bar
2 - User selects search
3 - System communicates with the server to retrieve products that contain the input string
4 - Sysem displays the products to the user

Alternative Flow(s)

NO PRODUCTS WITH STRING
